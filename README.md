# Repository Documentazione - Gruppo TheBitless

Questa repository contiene la documentazione ufficiale (sorgenti `.tex` e file `.pdf`) prodotta dal gruppo **TheBitless** per il progetto del corso di **Ingegneria del Software** (A.A. 2025/2026) presso l'Università degli Studi di Padova.

## Struttura del Repository

La cartella principale è organizzata per facilitare la consultazione dei documenti relativi alle diverse fasi del progetto:

- **candidatura/**: Contiene i documenti prodotti per la fase di candidatura.
  - **analisi-capitolati/**: Documento comparativo dei capitolati proposti, con valutazioni tecniche e motivazioni della scelta finale.
  - **dichiarazione-impegni/**: Documento formale con gli impegni assunti dal gruppo verso il progetto e il corso.
  - **lettera-di-presentazione/**: Lettera di presentazione del gruppo, con obiettivi, competenze e disponibilità operative.
  - **verbali/**: Archivio dei verbali redatti durante il progetto.
    - **verbali_esterni/**: Resoconti degli incontri con referenti di aziende proponenti.
    - **verbali_interni/**: Resoconti delle riunioni interne di coordinamento, pianificazione e avanzamento.
- **templates/**: Contiene le risorse grafiche e i file di stile condivisi per la redazione dei documenti.
